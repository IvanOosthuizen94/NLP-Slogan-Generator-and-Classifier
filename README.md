# NLP-Slogan-Generator-and-Classifier
A Deep Learning Natural Language Processing (NLP) project. I developed an LSTM neural network to generate creative business slogans based on specific industries, and trained a sequential classification model to accurately predict a company's industry based purely on its slogan text.

1. Navigate to the directory  
2. Launch Jupyter Notebook  
3. Open the Notebook: In your browser, click on capstone_project.ipynb to view the deep learning architecture, run the training cells, and generate your own slogans.  

### Project Structure
1. capstone_project.ipynb: The main Jupyter Notebook containing the Python code, text preprocessing, LSTM architecture, and classification model.  
2. slogan-valid.csv: The dataset containing real-world company slogans, industry classifications, and metadata used to train the neural networks.
